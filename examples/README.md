# Examples

Short end-to-end recipes for common use cases. Each folder contains a sample
`DESIGN.md`, a round-0 prompt, and notes on the expected outcome.

- [hero-landing/](hero-landing/) — cinematic hero for a B2B SaaS
- [pricing-page/](pricing-page/) — 3-tier pricing with playful personality
- [portfolio-section/](portfolio-section/) — case-study block with motion

Run each example from its folder:

```bash
cd examples/hero-landing
/picasso --design-loop "$(cat task.txt)"
```
